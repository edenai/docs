Eden AI infrastructure is hosted in secure data centers to ensure optimal performance and compliance.

## Primary Infrastructure

Eden AI's core infrastructure is located in **France (Lyon)**, ensuring:

- **GDPR Compliance** - Data processed within the European Union
- **Low Latency** - Optimized for European and global users
- **Data Sovereignty** - Your data stays within EU borders

## Provider Data Centers

When you make API calls through Eden AI, your data is processed by the underlying AI providers. Each provider operates their own data centers — locations vary per provider and model.

<Note>
To check the server location of a specific provider or model, refer to the [Eden AI model catalog](https://app.edenai.run/models).
</Note>

## Compliance

Eden AI supports various compliance requirements:

- **GDPR** - Full compliance for EU data protection
- **SOC 2** - Security controls certification
- **ISO 27001** - Information security management
