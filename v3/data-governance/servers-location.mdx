Eden AI infrastructure is hosted in secure data centers to ensure optimal performance and compliance.

## Primary Infrastructure

Eden AI's core infrastructure is located in **France (Paris)**, ensuring:

- **GDPR Compliance** - Data processed within the European Union
- **Low Latency** - Optimized for European and global users
- **Data Sovereignty** - Your data stays within EU borders

## Provider Data Centers

When you make API calls through Eden AI, your data is processed by the underlying AI providers. Each provider has their own data center locations:

| Provider | Primary Locations |
|----------|-------------------|
| OpenAI | United States |
| Anthropic | United States |
| Google | Global (configurable) |
| Amazon | Global (configurable) |
| Microsoft | Global (configurable) |

<Note>
Eden AI acts as a gateway and does not store your request/response data beyond what's needed for processing.
</Note>

## Compliance

Eden AI supports various compliance requirements:

- **GDPR** - Full compliance for EU data protection
- **SOC 2** - Security controls certification
- **ISO 27001** - Information security management
